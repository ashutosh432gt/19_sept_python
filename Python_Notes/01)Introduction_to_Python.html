Introduction to Python

Python

  • Python is a most popular,high-level,object oriented Programming language
  • High-Level:Python is easy to write and understand because it hides complex computer details.
  • Object-Oriented:It uses a style of coding that's good for organizing and reusing code.
  • Interpreted:You don't need to compile code; Python reads and runs it directly.
  • Interpreter:It's a program that executes Python code step by step, helping with errors and making it easy to work with.
  • Python was created by Guido van Rossum in 1991. It was developed by the Python Software Foundation and has become a widely used programming language known for its simplicity and versatility.

Features of Python

  • It is a High-Level language
  • It is simple and easy to understand programming language
  • It is case Sensitive programming language which means "a" is different from "A"
  • No datatype declaration is required

Applications of Python

  • Web Development
  • GUI-Graphical user Interface can be developed using Python
  • Business Applications
  • Games and 3D graphics
  • Accessing the database
