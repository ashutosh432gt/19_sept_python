Datatypes and variables

Datatypes

Dataypes specifies the kind of data a variable can hold.

Datatypes are further divied into two types

1)Primitive datatypes:-Primitive datatypes are predefined dataypes.For example--int,float,boolean,complex

2)Non-primitive datatypes:-Non-primitive datatypes are datatypes which is defined by users.For example-list,tuple,set

There are 6 standerd type of datatypes:-

  • Integers('int') :-56,23,58,72,-90.....all digits
  • Floating-point numbers ('float') :-26.34,65.21,76.11 ----all kinds of fractional numbers

variables

Variables refers to a container used to store data
