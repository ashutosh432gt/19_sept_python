CSS

CSS => cascading style sheet
there are three ways to implement css in web page

  1. Inline Css : to apply on specific page
  2. Internal Css : to apply on whole web page
  3. External Css : which is used to apply on website

Selectors

Element Selectors :
h1{
color:color name
}
Class Selectors
.classname{
color:color name
}
Group Selectors
h1,p{
color:color name
}
Id Selectors
#idname{
color:color name
}
Universal Selectors
*{
color:color name
}
